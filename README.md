# Breast Cancer Classification via DCE-MRI 🧬

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?logo=pytorch&logoColor=white)](https://pytorch.org/get-started/locally/)

Este repositorio contiene un pipeline de Deep Learning diseñado para la clasificación de lesiones de cáncer de mama utilizando imágenes de **Resonancia Magnética Dinámica con Contraste (DCE-MRI)**.

## 🚀 Descripción del Proyecto
El objetivo es procesar secuencias temporales de MRI para identificar biomarcadores de malignidad.

## 📁 Estructura del Repositorio
* `data/processed/`: CSVs con rutas y etiquetas.
* `src/`: Scripts modulares (`dataset.py`, etc.).
* `notebooks/`: Experimentos y preparación.

## 🛠️ Instalación
1. Clonar: `git clone https://github.com/PandoraRiot/breast-cancer-dce-mri-classification.git`
2. Librerías: `pip install -r requirements.txt`
